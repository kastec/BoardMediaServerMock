## BoardMediaServerMock (Node + TypeScript)

### Установка

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Сервер поднимется на порту `6010`: `http://localhost:6010`.

### Endpoint

- `POST /api/register` — обрабатывается методом `HelloController.register` и возвращает строку `HEllo`.
