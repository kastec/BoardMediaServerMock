1. Нужно создать простой сервер на NODE и typescript.    
Для сервера используем axios.  
2. Нудно добавить файл контроллера HelloController , в котором есть метоl register, который вернет сроку "HEllo".
3. Реристрация контролеров должна быть в файле routes.ts .
Этот файл додлжен собержать строку регистрации HelloController в виде
app.post('/api/register', HelloController.register) 
4. Приложение должно работать на порту 6010


npm install
npm run dev



Режим server (MasterTabletController)
   npm run dev -- server
   npm run dev -- server 8080
   npm run dev -- 8080 server
   
Обычный режим (SlaveController):
   npm run dev
   npm run dev -- 3000
   
   
   
Запуск BMS Proxy   
npm run dev -- server 6010    

запуск CT-master
npm run dev -- 5000

регистрация CT-мастера  
curl --location "http://localhost:6010/api/master-tablet/register" --header "Content-Type: application/json" --data "{\"payload\": {\"ip\": \"localhost:5000\"}}"

послать на CT через мастер   
curl --location --request GET "http://localhost:6010/api/proxy/api/slave/hello" --header "SEND-TO: localhost:5001" --header "Content-Type: application/json" --data "{\"key\": \"value\"}"
